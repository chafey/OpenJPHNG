# OpenJPHNG
Experimental repository for an OpenJPH refactoring

# Building

## Supported build environments:

| OS           | CPU     | Compiler
| ------------ | ------- | --------
| Mac OS X     | AArch64 | Apple Clang
| Linux Ubuntu | x86_64  | gcc
| Windows      | x86_64  | MSVC

## Requirements

* CMake 3.31 or newer
* C++ Build Environment (compiler, linker, make)

## Building

> ./build.sh
